Simulación de una red empresarial desarrollada en Cisco Packet Tracer.

*Características principales

- Segmentación mediante VLAN.
- Router-on-a-Stick.
- DHCP.
- SSH.
- ACL.
- Port Security.
- STP.
- Servidor DNS, HTTP, FTP y TFTP.

*Tecnologías 

- Cisco IOS
- VLAN
- IEEE 802.1Q
- DHCP
- SSH
- ACL
- Port Security
